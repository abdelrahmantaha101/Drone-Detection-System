# Drone-Detection-System
An AI-powered computer vision system for real-time drone detection, segmentation, and multi-object tracking using **YOLOv9** and **ByteTrack**.
## Overview

This project detects UAVs from RGB and Thermal (TIR) images or videos in real time. It provides accurate detection, segmentation, tracking, movement analysis, and intelligent alert generation.

## Features

- Real-time Drone Detection
- Drone Segmentation
- Multi-Object Tracking
- Live Radar Visualization
- Movement Direction & Distance Estimation
- Automatic Alert System
- Report Generation
- Downloadable Detection Results

## Technologies

- YOLOv9
- ByteTrack
- OpenCV
- Python
- CUDA

## Dataset

Anti-UAV RGBT Dataset

- RGB Images
- Thermal (TIR) Images
- 57.4K Samples
